# Account Management System

Project Overview
This is a simple **Account Management System** developed using Python, HTML, CSS, and JavaScript.  
The project demonstrates basic banking operations such as "creating an account, checking balance, depositing money, and withdrawing money".

# Technologies Used
- Python – Backend logic and simple server
- HTML – User interface pages
- CSS – Styling the web pages
- JavaScript – Client-side interactions
- Git & GitHub– Version control

# Project Structure
Account management/
- accounts.txt # Stores account data
- bank_account_management.py # Core account logic
- simple_server.py # Simple backend server
- index.html # Entry page
- home.html # Home/Dashboard page
- create_account.html # Create new account page
- check_balance.html # Check account balance
- deposit_money.html # Deposit money page
- withdraw_money.html # Withdraw money page
- style.css # Common styles
- app.js # JavaScript logic
- static/ # Static assets (if any)

- Create a new bank account
- Check account balance
- Deposit money
- Withdraw money
- Store account details in a text file
